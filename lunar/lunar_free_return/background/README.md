# Background

This folder contains the research question, literature context, and prior results summaries.

## The 4-Part Structure

Background should include the question being tested at this iteration, what has been done in the area, and what's known about the question from the literature. This provides the context needed for forming the hypothesis.

## Files

- `01_initial_question.md` - Your starting research question and initial literature context
- `iteration_XX_background.md` - Background updates for successive iterations (smaller, mostly restatements plus new context)

## Note on Literature

AI often won't have good access to literature, and may be able to provide
very general statements about where and how something has been tried before.
This can include both false negatives - the AI says something hasn't been done
before when it actually has been - and false positives - the AI says that
something has been done when it hasn't. It might be very confident about that BTW.

Verify important literature claims independently.
